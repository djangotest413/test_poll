Разворачивается локально, файл с требованиями прилагается.
Протестировать можно штатными стредсвами restframework.
Авторизация закоментирована для простоты тестирования.

http://127.0.0.1:8000/token/ - получить токен
http://127.0.0.1:8000/token/refresh/ - обновить токен
http://127.0.0.1:8000/poll/ - добавить опрос
http://127.0.0.1:8000/poll/<int:pk>/ - обновить и удалить опрос (pk = id опроса)
http://127.0.0.1:8000/poll/question/ - добавить вопрос к опросу
http://127.0.0.1:8000/poll/question/<int:pk>/ - обновить и удалить вопрос
http://127.0.0.1:8000/poll/active/ - получить список активных опросов
http://127.0.0.1:8000/questions/<int:pk>/ - получить список вопросов по опросу (pk = id опроса)
http://127.0.0.1:8000/answer/ - добавить ответ
http://127.0.0.1:8000/polls/<int:user_id>/ - список пройденых опросов с вопросами и ответами(user_id = user_id пользователя)
